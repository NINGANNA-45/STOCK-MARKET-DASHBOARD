# STOCK-MARKET-DASHBOARD
A web-based stock market dashboard that displays stock prices, market trends, charts, and financial data in an interactive and user-friendly interface.

This project helps users:
Track stock prices
Analyze trends using charts
Monitor multiple stocks in one place

Tech Stack:
HTML/CSS/JAVASCRIPT
Custom CSS
Charts.js

The Stock Market Dashboard demonstrates the practical implementation of frontend development concepts. It highlights the ability to build interactive, data-driven web applications and showcases skills relevant to modern web development roles.
